# 储能逆变器运维规范

## 1. 适用范围

本规范适用于 50kW / 100kW / 250kW 组串式储能逆变器日常巡检与故障处置。

## 2. 设备型号对照

| 型号 | 额定功率 | 最大直流电压 |
|------|----------|--------------|
| INV-50K | 50kW | 1100V |
| INV-100K | 100kW | 1100V |
| INV-250K | 250kW | 1500V |

## 3. 日常巡检项目

1. 检查直流侧绝缘电阻，应 ≥ 1MΩ
2. 检查交流侧接线端子温升，应 < 65℃
3. 检查风扇运行状态，无异响、无停转
4. 核对固件版本，当前推荐版本 `V3.2.1`

## 4. 常见故障代码

| 故障码 | 含义 | 处置建议 |
|--------|------|----------|
| E101 | 直流过压 | 检查 BMS 充电策略，降低充电功率 |
| E203 | IGBT 过温 | 清理进风口滤网，检查环境温度 |
| E305 | 绝缘阻抗低 | 停机排查直流侧线缆绝缘 |
| E410 | 通信中断 | 检查 CAN/RS485 接线与终端电阻 |

## 5. 安全注意事项

- 维护前必须断开交直流开关并挂牌上锁
- 严禁在雷雨天气进行室外柜体开盖作业
- 更换功率模块后需重新执行绝缘测试
